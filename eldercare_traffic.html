接送專車交通路線

中心提供專車定點接送服務，但因考量車次及接送路線安排，定點搭車及抵達中心的時間不固定。
發車時間：上午7點40分、下午3點20分

由松柏中心出發 → 行經南門路 → 右轉健康路 → 直行經林森地下道，此處為停車定點 → 接林森路、崇善路口，此處為停車定點 → 林森路直行接東門路二段右轉 → 左轉裕農路，此處為停車定點 → 裕農路直行 → 左迴轉東寧路 → 直行東寧路 → 右轉林森路，經林森路口小公園，此處為停車定點 → 林森路小東路口，此處為停車定點 → 林森路直行接東豐路左轉 → 右轉長榮路三段 → 直走長榮路四段，此處為停車定點 → 經開元路，接長榮路五段，此處為停車定點 → 經長榮地下道、長榮路五段，此處為停車定點 → 右轉公園路 → 左轉北安路 → 左轉接西門路四段，此處為停車定點 → 右轉和緯路二段，此處為停車定點 → 直行接和緯路三段 → 左轉文賢路，此處為停車定點 → 接成功路右轉金華路四段 → 金華路四段，此處為停車定點 → 左轉民生路二段 → 右轉西門路二段，此處為停車定點 → 左轉西門路健康路口，此處為停車定點 → 直行健康路 → 右轉南門路 → 返回松柏中心
